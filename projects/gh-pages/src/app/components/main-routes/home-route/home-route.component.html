ng-ui2qp

Angular library that synchronizes UI component's state with the Browser's query params, improving the user experience allowing the creation of shareable links.

Frictionless

Was built for Angular. Start from scratch or drop into your existing applications.

Easy to use

ng-ui2qp required almost no setup and can be adjusted to different use cases through configuration provided at different levels. Extends Angular ReactiveForms which makes it familiar and intuitive.

Declarative

The library API provides a declarative approach to interact with its many features.

Well documented

All developers appreciate a well documented tool that's why we've spent hours documenting it an including many examples, suggestions , best practices etc. If you feel something can be improve please drop us an issue in github.

It's free!!

ng-ui2qp it's free and open source, we love the open source community and this is how we contribute to it. the source code can be found on Github, fork it or help us to improve it.
